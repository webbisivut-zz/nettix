Contributors: WebData Oy
Donate link: https://webdata.fi/
Tags: wordpress, plugin, nettix
Requires at least: 3.9
Tested up to: 5.8.3
Stable tag: 1.4.4

This plugin fetches data from Nettix API and displays the data on frontend. Use shortcode [nettix] to display the vehicles. Cars and motorbikes are currently supported.
